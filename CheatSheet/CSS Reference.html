CSS Description

Tag Name Description
align-content Align Content Specifies the alignment between the lines inside a flexible container when the items do not use all available space.
align-items Align Items Specifies the alignment for items inside a flexible container.
background-color Background Color Specifies the background color of an element.
background-size Background Size Specifies the size of the background images.
color Color Sets the color of text.
font-family Font Family Specifies the font family for text.
font-size Font Size Specifies the font size of text.
font-style Font Style Specifies the font style for text.
font-weight Font Weight Specifies the weight of a font.
height Height Sets the height of an element.
justify-content Justify Content specifies the alignment between the items inside a flexible container when the items do not use all available space.
left Left Specifies the left position of a positioned element.
margin Margin Sets all the margin properties in one declaration.
margin-bottom Margin Bottom Sets the bottom margin of an element.
margin-left Margin Left Sets the left margin of an element.
margin-right Margin Right Sets the right margin of an element.
margin-top Margin Top Sets the top margin of an element.
padding Padding A shorthand property for all the padding properties.
padding-bottom Padding Bottom Sets the bottom padding of an element.
padding-left Padding Left Sets the left padding of an element.
padding-right Padding Right Sets the right padding of an element.
padding-top Padding Top sets the top padding of an element.
right Right specifies the right position of a positioned element.
text-align Text Align Specifies the horizontal alignment of text.
text-indent Text Indent Specifies the indentation of the first line in a text-block.
text-justify Text Justify Specifies the justification method used when text-align is "justify".
top Top specifies the top position of a positioned element.
vertical-align Vertical Align Sets the vertical alignment of an element.
white-space White Space Specifies how white-space inside an element is handled.
widows Widows Sets the minimum number of lines that must be left at the top of a page when a page break occurs inside an element.
width Width Sets the width of an element.
